# formula-based-normal-map-generator

A simple Python tool that generates grayscale height maps and normal maps from math equations.



Self-use tool, but you're welcome to use it if you want to make 3D objects into accurate normal maps and don't want to use complex software like blender.


## Usage


Replace the `height_function` with mathematical expressions of your image that calculate the height of every pixel.


The script demonstrates an example of computing the height of a space station module shape.
